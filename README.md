# Yoolax Zigbee Module

Adds support for Yoolax shades with a Zigbee module.

# Prerequisites
- HomeyPro is plugged in, and connected to 2G Wifi
  ## Laptop/Device Requirements
  - Your device must be on the same 2G Wifi network (NOT an isolated/guest network), or plugged directly into HomeyPro via USB
  - Nodejs installed on your Laptop/Device
  - HomeyCLI installed on your Laptop/Device

# Installation
  ## On your Laptop/Device
  - Follow instructions here to set up NodeJS, and HomeyCLI on your device  
      [https://apps.developer.homey.app/the-basics/getting-started](https://apps.developer.homey.app/the-basics/getting-started)
  - clone this repo  
    ```git clone https://github.com/givemelove/com.virtualmood.yoolax```
  - enter that new folder (recently clone) and run the app
    ```
    cd com.virtualmood.yoolax
    homey login
    homey app validate
    homey app run
    ```
